Flow

How Sentinel-MCP Operates

1. Client Connects

Claude, ChatGPT, Copilot, Cursor, Continue, and others register and authenticate with OAuth 2.1.

2. Capabilities Are Scoped

Each client receives only the abilities granted in its allowlist and operates under rate limits.

3. Abilities Execute

Content, media, taxonomy, diagnostics, SEO, multilingual, and WooCommerce reads run against live WordPress state.

4. Everything Is Auditable

Activity logs capture who called what, status, timing, and IP for operations visibility.

Get Started

Install In Minutes

  1. Install and activate Sentinel-MCP in WordPress.
  2. Open Settings -> Sentinel-MCP and run the Get Started wizard.
  3. Create or verify OAuth client setup.
  4. Copy the MCP server URL from the Connect/Status area.
  5. Add that URL to your AI client and authorize once.
  6. Begin natural-language operations against your WordPress site.
